Javascript

Primitives

String: a set of characters, which each correspond to a 16-bit unsigned integer. Strings are immutable in JS, which means they cannot be changed after creation.
To initilize a string, use single or double quotiation marks or backticks.

s = 'Sinlge quotiation marks';
s = "Double quotiation marks";
s = `backticks`;

Number: holds integers and floats as double-precision 64-bit binary value. Very large integers can be stored in the BigInt type.

num = 10;
num_as_string = "10";

// parsing string to Number
parseInt(num_as_string); // with parseInt()
Number(num_as_string) // with typecast

Boolean: can be either true og false. Often used to determine if a block of code should be run on not.

b = true;
if (b) {
    console.log("b is true!");
} // expected output: "b is true!"

Null & undefined: Null has only value, which is null. It can be used to indicate that a variable points to no object. Undefined means that a variable has not been assigned a value, thus the value is undefined.

let variable;
console.log(variable); // exprected output: undefined

Objects & datastructures

Objects: can be seen as a collection of properties. Properties can be added as a "key" with a corresponding "value". These values can be of any type.

const o = {}; // initialization of empty object
o.id = "123"; // adding "id" property with value "123"
const person = {name: "John"} // object initialization using curly-braces.
person.age = 25; // adding new property "age"
delete person.name; // deleting "name" property
person.o = o;
console.log(person); // expected output: {name: "John", o: {â€¦}}

Arrays: is an object that can contain multiple values of any type. They are zero-indexed, meaning the first element of an array has index 0. Elements in an array can be added, deleted and accessed using built-in functions. We can also loop the array with a for/foreach loop and use lambdas such as .filter(), .map() and .reduce()

const friends = [];
const john = {
    name: "John",
    age: 22
};
friends.push(john); // adds element to end of array
friends.pop();  // removes last element
friends.slice();  // creates a copy of the array
friends[0]  // get the first element

let, const & var

let declares a block-scoped local variable. var declares a variable in the global-scope. Finally const declares a constant value in the block-scope.

{   // this is a new scope block
    let scopeVar = "abc";
    {
        // this is a nested scope that inherits from the prevoius scope
        scopeVar = 123;
    }
    console.log(scopeVar); // output: "abc"
}

{
    let scopeVar = "abc";
    {
        let scopeVar = 123;
        console.log(scopeVar);  // output: 123
    }
    console.log(scopeVar);   // output: "abc"
}

Variables can be used within string literals, using backticks and ${} for an expression. By using this it is possible to output complex data through strings.

const person = {firstName: "John", lastName: "Johnson", age: 22};
console.log(`Hello! My name is ${person.firstName} ${person.lastName} and I'm ${person.age} years old!`);
// output: "Hello! My name is John Johnson and I'm 22 years old!";

Functions

Functions are a set of statements that perform some task such as calculating a value. They take some input and return an output.

// function definition / declaration
function myFirstFunction(message) {
    return "Your message: " + message;
}
console.log(myFirstFunction("Hello!"));  // output: "Your message: Hello!"

Function expressions & Arrow functions

It is possible to define an anonymous function inside an expression. For example you can assign an anonymous function to a variable or constant. Another way of using this concept is known as arrow funtions, which is a shorthand / more compact way of defining a function expression (though more limited). Furthermore arrow functions / anonymous functions are often used as callbacks, which is described in the next section.

const x = function() {} // defining an anonymous function and assigns it to x
const myArrowFunction = () => {
    console.log("Hello from the anonymous function which is also an arrow function");
}

// Comparing function to arrow function
function (a) {
    return a + 100;
}

// Breaking down to Arrow function equivalent
(a) => {
    return a + 100;
}

(a) => a + 100;
a => a + 100;

// Can also be with no arguments
let a = 10;
let b = 20;
function() {
    return a + b + 100;
}

() => a + b + 100;
const func = () => a + b + 100;  
// combining function expression, anonymous function and arrow function

Callback functions

Callback function is a function that is passed into another function as an argument, which is then invoked inside the outer function. It is often used to continue code execution after an asynchronous operation has completed (asynchronous callback).

// example using callback functions
function interact(genericInteraction, name) {
    console.log(genericInteraction(name));
}

const poke = (name) => { 
    return `${name} has been poked!`; 
}

const impeach = (name) => {
    return `${name} has been impeached for the 2nd time!`;
}

interact(poke, "Anders");
interact(impeach, "Donald Trump");
// output: 
"Anders has been poked!"
"Donald Trump has been impeached for the 2nd time!"

Loops

In JS there are for, for-each, for-in and for-of loops, which we can use on objects, arrays etc. But we aim to use functional approaches wheen looping such as .map(), .reduce(), .find() .filter() and the like.

.map() takes a callback that is performed on all elements, then returns a new array with the result.
.filter() takes a predicate function and returns the elements of an array that meet the condition specified in a callback function.
.reduce() calls the specified callback function for all the elements in an array. The return value of the callback function is the accumulated result, and is provided as an argument in the next call to the callback function.
.find() returns the value of the first element in the array where predicate is true, and undefined otherwise. 

// short example using .map() to log index and value
const failures = ["misclick", "falling down", "skateboard accident", "tiktok fail"];

failures.map((failure, index) => {
    console.log(index, failure);
});
